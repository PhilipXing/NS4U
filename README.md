## 🚀 Getting Started

### Windows
1. Download the latest Windows version from [ns4u.fun](https://ns4u.fun)
2. Run the installer
3. Launch NS4U and start playing!

### Android
1. Download the APK from [ns4u.fun](https://ns4u.fun)
2. Enable "Install from Unknown Sources" in settings
3. Install and enjoy!

### macOS
1. Download the macOS version from [ns4u.fun](https://ns4u.fun)
2. Open the DMG file
3. Drag NS4U to Applications folder

### iOS
1. Download from [ns4u.fun](https://ns4u.fun)
2. Follow the installation guide
3. Trust the developer certificate in Settings
